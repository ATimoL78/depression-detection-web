## 📝 详细部署步骤 (跟着做就行)

### 第1步: 注册Railway账号

1. 打开浏览器,访问: **https://railway.app**
2. 点击右上角的 **"Login"** (登录)
3. 选择 **"Login with GitHub"** (使用GitHub登录)
4. 如果没有GitHub账号:
   - 先去 https://github.com 注册一个GitHub账号
   - 然后回来用GitHub登录Railway

### 第2步: 上传项目到GitHub

1. 访问: **https://github.com/new** (创建新仓库)
2. 填写信息:
   - Repository name(仓库名): `depression-detection-web`
   - Description(描述): `抑郁症检测系统`
   - 选择 **Public** (公开) 或 **Private** (私有)
3. 点击 **"Create repository"** (创建仓库)
4. 按照页面提示上传您的项目文件

**如果不会用GitHub,可以联系我帮您上传!**

### 第3步: 在Railway部署

1. 登录Railway后,点击 **"New Project"** (新建项目)
2. 选择 **"Deploy from GitHub repo"** (从GitHub仓库部署)
3. 如果是第一次,需要点击 **"Configure GitHub App"** 授权Railway访问您的GitHub
4. 选择您刚才创建的仓库: `depression-detection-web`
5. Railway会自动开始部署,等待3-5分钟

### 第4步: 获取网站地址

1. 部署完成后,点击项目
2. 点击 **"Settings"** (设置)
3. 找到 **"Domains"** (域名)
4. 点击 **"Generate Domain"** (生成域名)
5. 复制生成的域名,例如: `https://depression-detection-production.up.railway.app`

### 🎊 完成!

现在您可以访问这个域名,您的网站就永久部署好了!
