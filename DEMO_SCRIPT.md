### 第2部分: 假笑检测演示 (4分钟)

**演示词:**
> "首先演示我们的核心功能之一 - 假笑检测。基于Paul Ekman的Duchenne Smile理论,系统可以区分真笑和假笑,准确率达到86%。"

**操作步骤:**

1. **进入检测页面**
   - 点击"实时智能识别"
   - 点击"启动扫描"
   - 允许摄像头权限

2. **演示真笑(Duchenne Smile)**
   - 对着摄像头自然微笑
   - 确保眼睛周围有笑纹(AU6激活)
   - 观察右侧假笑分析卡片
   
   **预期结果:**
   ```
   ✅ 真实笑容 (绿色边框)
   Duchenne微笑: 是
   AU6/AU12比值: 0.75
   左右对称性: 85%
   真实性置信度: 90%
   ```

3. **演示假笑(Social Smile)**
   - 仅嘴角上扬,眼睛不笑
   - 保持眼睛平静
   - 观察检测结果变化
   
   **预期结果:**
   ```
   ⚠️ 假笑检测 (红色边框)
   Duchenne微笑: 否
   AU6/AU12比值: 0.25
   原因: "AU12 active but AU6 missing (no eye involvement)"
   ```

**演示词:**
> "看,当我真心微笑时,系统显示绿色边框,这是真实的Duchenne微笑。而当我只是礼貌性地嘴角上扬,眼睛不笑时,系统立即检测到这是假笑,显示红色边框。这对于识别情绪伪装非常有用。"

**操作步骤:**

   - 点击麦克风按钮 🎤
   - 允许麦克风权限
   - 观察语音情绪面板出现

2. **演示快乐情绪**
   - 用欢快的语气说话

   **预期结果:**

   能量: 0.35

3. **演示悲伤情绪**

   **预期结果:**
   ```
   语音情绪: sad
   置信度: 70%
   音高: 140 Hz
   能量: 0.18
   ```

**演示词:**

### 第4部分: 多模态情绪融合演示 (3分钟)

**演示词:**
> "最强大的功能是多模态情绪融合。系统会同时分析您的面部表情和语音情绪,当两者一致时,置信度会大幅提升;当不一致时,系统会预警可能存在情绪压抑或伪装。"

**操作步骤:**

1. **演示一致性高的情况**
   - 面部微笑 + 欢快语气
   - "我今天真的很高兴!"
   
   **预期结果:**
   ```
   面部情绪: happy (85%)
   语音情绪: happy (80%)
   → 融合结果: happy (95%) ✅
   来源: multimodal-consistent
   ```

2. **演示情绪不一致的情况**
   - 面部微笑(假笑) + 悲伤语气
   - "我很好,真的..."(低沉语气)
   
   **预期结果:**
   ```
   面部情绪: happy (70%)
   语音情绪: sad (75%)
   → 融合结果: sad (64%) ⚠️
   来源: multimodal-voice-dominant
   冲突: 面部和语音情绪不一致,可能存在情绪压抑或伪装
   ```

**演示词:**
> "注意看,当我的面部表情和语音情绪一致时,系统的置信度达到95%。而当我试图用笑容掩饰悲伤时,系统立即检测到不一致,并预警可能存在情绪压抑。这对于发现隐藏的心理问题非常重要。"

## Q&A环节

**Q1: 系统的准确率如何?**
> A: 假笑检测86%,多模态融合95%,已达到国际领先水平。

**Q2: 是否需要专业设备?**
> A: 不需要,普通摄像头和麦克风即可,建议720p以上。

## 演示注意事项

### 准备工作
- ✅ 提前测试摄像头和麦克风
- ✅ 确保光线充足
- ✅ 准备备用浏览器
- ✅ 清除浏览器缓存

### 演示技巧
- 💡 面对摄像头,保持面部完整
- 💡 说话清晰,避免背景噪音
- 💡 表情自然,不要过度夸张
- 💡 提前练习假笑和真笑的切换

### 应急预案
- 🔧 摄像头无法启动 → 检查权限,刷新页面
- 🔧 麦克风无法启动 → 检查权限,更换浏览器
- 🔧 检测不准确 → 调整光线,重新校准
- 🔧 系统卡顿 → 关闭其他标签页,重启浏览器
